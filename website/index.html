How it works

Four steps between an OpenAPI spec and a tool your AI agent can use.

  1. 1

    Import

    Paste an OpenAPI/Swagger URL or upload a YAML/JSON file in the Toolize UI.

  2. 2

    Parse

    Toolize reads every endpoint, schema, and authentication requirement.

  3. 3

    Generate

    Each endpoint becomes an MCP tool, served as JSON-RPC 2.0 on /mcp.

  4. 4

    Connect

    Point Claude Desktop, Cursor, or any MCP client at that URL.

Get started in under a minute

Just one dependency: Docker.

terminal 
docker build -t toolize .
docker run -p 8080:8080 -v toolize-data:/data toolize

Then open your browser at:

url 
http://localhost:8080

Import an OpenAPI/Swagger spec, and the generated MCP tools are immediately available at:

url 
http://localhost:8080/mcp
