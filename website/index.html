OpenAPI → MCP

Transformez n'importe quelle API
en outils MCP, instantanément.

Toolize importe une spécification OpenAPI et expose immédiatement chaque endpoint comme un outil MCP — sans écrire une ligne de code, sans redémarrer quoi que ce soit.

Démarrer avec Docker Voir le code source

C'est quoi, Toolize ?

Les assistants IA (Claude, Cursor, et tout autre client compatible MCP) parlent MCP (Model Context Protocol), pas OpenAPI. Toolize comble ce fossé : importez une spec, obtenez des outils MCP prêts à l'emploi.

📥

Import en un clic

Collez une URL Swagger/OpenAPI ou uploadez un fichier YAML/JSON. Toolize s'occupe du parsing.

⚙️

Génération automatique

Chaque endpoint de l'API devient un outil MCP, avec ses paramètres et son authentification.

⚡

Zéro code, zéro redémarrage

Les outils sont disponibles immédiatement sur /mcp, dès la fin de l'import.

Comment ça marche

Quatre étapes entre une spec OpenAPI et un outil utilisable par votre agent IA.

  1. 1

    Importer

    Renseignez l'URL d'une spec OpenAPI/Swagger ou déposez un fichier YAML/JSON dans l'interface Toolize.

  2. 2

    Parser

    Toolize analyse les endpoints, schémas et éventuels mécanismes d'authentification de l'API.

  3. 3

    Exposer

    Chaque endpoint devient un outil MCP, servi en JSON-RPC 2.0 sur /mcp.

  4. 4

    Connecter

    Pointez Claude Desktop, Cursor, ou tout client MCP compatible Streamable HTTP vers cette URL.

Démarrage rapide

Une seule dépendance : Docker.

terminal 
docker build -t toolize .
docker run -p 8080:8080 -v toolize-data:/data toolize

Puis ouvrez votre navigateur sur :

url 
http://localhost:8080

Importez une spec OpenAPI/Swagger, et les outils MCP générés sont immédiatement disponibles sur :

url 
http://localhost:8080/mcp
